## 01. 작업 요약

- 결과: 완료
- 소요 시간: 단기 핫픽스

## 02. 작업 로그

- 복구 화면이 `loading` 하나를 기존 연결 확인/다른 Google 로그인/새 연결 생성에 공유하는 구조 확인
- `recoveryChecking`, `creatingConnection` 상태를 분리
- 기존 연결 확인은 `확인 중...`으로만 표시
- 새 연결 진행은 `새 연결 준비 중...`으로 별도 표시
- 복구 확인과 invite 생성에 8초 타임아웃 추가
- 타임아웃/실패 시 화면에 오류 문구 표시
- 새 연결 버튼 글자 대비를 `bg-primary text-on-primary`로 수정
- Log 페이지에 핫픽스 내역 추가
