## 04. 구현 결과
✅ 완료 항목:
- 채팅창에 파일 드래그앤드랍 가능

⚠️ 미완료 항목:

## 05. 특이점 / 결정사항
- 파일을 드롭하자마자 즉시 전송하지 않고, 기존 전송 확인 UI를 재사용한다.
- 여러 파일을 한 번에 드롭한 경우 현재는 첫 번째 파일만 첨부한다.
