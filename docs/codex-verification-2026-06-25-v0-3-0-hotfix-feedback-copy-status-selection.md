## 04. 구현 결과

✅ 완료 항목:

- 미완료 리포트 전체 텍스트를 클립보드로 복사
- 복사 텍스트에 작성자, 타입, 시간, 본문 포함
- `done` 상태 리포트는 복사 대상에서 제외
- 완료 버튼을 누르면 상태 chip이 완료로 바뀌고 복사 대상 수에서 제외
- 상태 편집 선택 버튼을 색상 반전, 확대, ring, shadow로 강조
- 고대비 모드에서도 선택 버튼을 흰 배경 + 검은 글씨 + 노란 ring으로 표시

⚠️ 미완료 항목:

- 실기기 클립보드 권한 확인은 배포 후 필요

## 05. 특이점 / 결정사항

- “전체 내용 복사”는 완료되지 않은 `open` 리포트만 대상으로 한다.
