## 05. 특이점 / 결정사항
- `바쁨`, `혼자만의 시간`, `공부중`, `독서중`은 상태 태그가 아니라 빠른 상태 메시지로만 유지했다.
- 플레이어 상태는 커플 공용 데이터가 아니라 기기별 사용 선호이므로 Firestore가 아닌 localStorage에 저장했다.
- 오디오 `load()` 중 발생하는 자동 `pause` 이벤트가 사용자 중지 상태로 저장되지 않도록 방지했다.
