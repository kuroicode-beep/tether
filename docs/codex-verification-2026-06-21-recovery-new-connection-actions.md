## 05. 특이점 / 결정사항
- 기존 `handleStartNewConnection()`은 초대 코드 생성 전용으로 유지했다.
- 새 `handleJoinNewConnection()`은 입력 코드를 초기화하고 `join` 단계로 이동만 수행한다.
- 이 수정은 연결/알림 토큰 로직에는 영향을 주지 않는다.
