# Tether v0.4.3 PRD

작성일: 2026-06-25  
버전: v0.4.3  
대상: Tether PWA

## 01. 제품 개요
Tether는 커플이 채팅, 상태, 교환일기, 사진, 콘텐츠, 파일, 링크, 데이트 기록을 한 공간에서 공유하는 PWA 앱이다. v0.4.x의 핵심 목표는 기존 커플 커뮤니케이션 기능을 유지하면서 공유 자료와 기록 기능을 확장하고, 저시력/고대비 사용성을 안정화하는 것이다.

## 02. 목표
- 채팅에서 오가는 파일과 음악을 별도 자료실에서 다시 찾을 수 있게 한다.
- 콘텐츠 기록에 URL과 이미지를 함께 저장할 수 있게 한다.
- 링크공유와 데이트 레시피를 각각 독립 메뉴로 제공한다.
- 사진첩 목록에서 코멘트를 바로 확인할 수 있게 한다.
- 상태 표시와 편집 경험을 분리해, 메인 화면은 읽기 쉽고 편집창은 선택 상태가 명확하게 보이게 한다.
- 설정에서 상대방 닉네임은 상대가 설정한 값만 보여준다.
- 알림음을 여러 선택지로 제공한다.

## 04. 주요 기능 범위
### 04-1. 커플 공유
- Google 로그인 기반 계정 연결
- 커플 연결/복구
- 연결된 사용자 간 채팅, 일기, 사진, 콘텐츠, 자료 공유

### 04-2. 채팅
- 텍스트, 이미지, 일반 파일, 음악 파일 전송
- 음악 파일은 채팅 말풍선에서 바로 재생
- 일반 파일은 열기/다운로드로 연결
- 상대가 읽기 전에는 내 메시지 옆에 읽기 전 아이콘 표시
- 상대가 읽으면 아이콘 숨김

### 04-3. 자료실
- 채팅에서 전송된 일반 파일/음악 파일을 자동으로 모아 표시
- 음악 파일은 자료실에서도 바로 재생
- 일반 파일은 외부 프로그램 또는 브라우저 다운로드로 연결

### 04-4. 링크공유
- 사이트명, URL, 요약 설명 저장
- 선택적으로 바로가기 파일 업로드
- 자료실과 분리된 독립 메뉴

### 04-5. 데이트 레시피
- 날짜별 같이 먹은 음식 기록
- 간단한 한 줄 로그 중심
- 링크공유/자료실과 분리된 독립 메뉴

### 04-6. 콘텐츠
- 책, 영화, 드라마, 유튜브, 기타 콘텐츠 기록
- URL 추가
- 이미지 직접 업로드
- 상태, 별점, 감상 관리

### 04-7. 사진첩
- 사진 업로드
- 등록 코멘트 저장
- 목록 카드 하단에 코멘트 표시

### 04-8. 상태
- 표정 아이콘, 상태 태그, 상태 메시지 표시
- 상태 변경은 편집 후 확정 버튼으로 저장
- 메인 상태 태그는 고대비에서 검은 배경/흰 글씨/흰 라인 유지
- 편집창 선택 태그는 반전 스타일로 명확히 표시

### 04-9. 설정
- 내 닉네임과 프로필 사진 수정
- 상대 닉네임은 읽기 전용
- 알림 토큰 재등록 및 테스트 알림
- 알림음 선택: 물방울, 차임, 반짝, 포근벨, 톡톡, 무음

## 05. 비범위
- URL 자동 썸네일 추출
- 링크/데이트 레시피 수정·삭제 UI
- 완전한 네이티브 앱 기능
- 웹 푸시의 iOS 시스템 알림음 커스터마이징

## 06. 성공 기준
- v0.4.3 빌드가 통과한다.
- Firebase Hosting 배포가 완료된다.
- 홈에서 자료실, 링크공유, 데이트 레시피가 각각 별도 메뉴로 보인다.
- 설정에서 상대 닉네임을 편집할 수 없다.
- 고대비 모드에서 메인 상태 태그가 명확히 읽힌다.
- 알림음 6개 옵션이 설정 화면에 표시된다.

## 07. 리스크
- iOS PWA는 브라우저 정책상 일부 포커스/푸시 동작이 제한될 수 있다.
- URL 자동 미리보기는 CORS 제약 때문에 Cloud Function 또는 외부 API가 필요하다.
- 기존 E2E 스크립트는 Google-only 가입 정책과 충돌해 bootstrap 재설계가 필요하다.
